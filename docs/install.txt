Vedro System
Инструкция по установке.


Для работы CMS Vedro System необходимо:
 - Web-server Apache 1.x/2.x (возможно работает и с другими Web-server-ами, но это не проверено);
 - PHP 5.x;
 - MySQL 4.x/5.x.


Установка производится помещением содержимого архива в некоторую директорию, и затем настройкой Web-server-a.

На этапе настройки Web-server-а необходимо корректно настроить виртуальный хост (либо по какому-нибудь доменному имени, либо по определённому порту (ниже приводится пример настройки по порту)).

Чтобы настроить виртуальный сервер по порту требуется:
Открыть конфигурационный файл httpd.conf/apache2.conf и добавить в него строку

	Listen *:81 # 81 — порт на котором будет слушать Web-server.

Данную строку необходимо добавить после "Listen *:80", если таковая имеется.
Затем необходимо открыть файл конфигурации виртуальных хостов (httpd-vhosts.conf) и добавить в него примерно следующие строки, заменив путь до директории в которую было распаковано содержимое архива.

	NameVirtualHost *:81
	<VirtualHost *:81>
		DocumentRoot /usr/local/www/vedro
		<Directory "/usr/local/www/vedro">
			DirectoryIndex index.php
			ErrorDocument 404 /404.php
		</Directory>
		ErrorLog /usr/local/www/vedro/logs/http-error_log
		CustomLog /usr/local/www/vedro/logs/http-access_log common
	</VirtualHost>

Очень важной здесь является строка

	ErrorDocument 404 /404.php

которая обеспечивает перенаправление ненайденных страниц на файл /404.php

Также необходимо настроить PHP. Открыв php.ini (конфигурационный файл пхп) найдите там строчку "output_buffer=". И если значение которому равно - "off", то установите значение равным 4096. Если указано какое-нибудь другое числовое значение - оставьте как есть ;)


После этого нужно выполнить на сервере БД скрипт /sqls/vedro_db_and_tables.sql, который создаст базу данных "vedro", а также таблицы в ней, необходимые для работы Vedro System.

	mysql -h host -u user -ppassword < vedro_db_and_tables.sql


Затем, нужно дать права на запись вашему Web-серверу для директорий /cache, /logs и /users


Далее необходимо отредактировать конфигурационный файл самой Vedro System - /config.php. И указать в нём логин/пароль для доступа в БД, а также другие необходимые настройки.


Затем, требуется установить модули, через которые Vedro System будет "общаться" с пользователями.
Делается это следующим образом:
В командной строке перейти в директорию с распакованной Vedro System. Отредактировать скрипт "mod", указав в начале путь до php (-cli), или же, если у вас Windows вообще убрать первую строку. Затем, набрать команду

	./mod -i modules/system_admin/

эта команда установит модуль администрирования системы через Web-интерфейс*.

Если вы работаете в операционной системе Windows, вам необходимо сделать так, чтобы скрипт mod запускался интерпритатором языка PHP (PHP-CLI).
Если вы работаете в операционной системе *nix и при запуске операционная система "говорит": "в доступе отказано", вам необходимо дать права на запуск файлу mod.


Всё. Установка и настройка Vedro System на этом закончилась.
Логин/Пароль для доступа в установленную систему - root/pass


* К сожалению, в данный момент, модуль администрирования системы не имеет возможности устанавливать другие модули через Web-интерфейс. Поэтому, если вам потребуется установить другие модули, то их установку необходимо производить через командную строку.
